Math 102 wk 01 - Section 01.02 Linear Equations and Rational Equations

Linear Equations

   Linear Equation in One variable - equation that can be written Ax + B = 0, if A & B are real numbers and A != 0  
       the value that solves the equation is a solution or ??root??

   Solution Set - set of all solutions that satisfy the equation
    
   Equivalent Equations - 2 or more equations that have the same solution set


Solving Linear Equations

    1) Simplify algebraic expresson on each side by removing grouping symbols and combining like terms
    2) Collect all variables on one side and numbers or constant terms on the other side
    3) Isolate the variable and solve
    4) Check the proposed solution in the original equation

  ex: 4x + 5 = 29
         - 5   -5

      4x = 24
      /4   /4

      x = 6  

  ex02: 4(2x+1) = 29 + 3(2x - 5) 
         8x + 4 = 29 + 6x - 15
             2x = 10
              x = 5   

  ** in this example 'clear the equation of fractions' by multiplying both sides by the least common denominator **           
  ex03: (x-3)/4 = 5/14 - (x+5)/7    -- use LCD of 28 to multiply all items
         7(x-3) = 10 - 4(x+5)
        7x - 21 = 10 - 4x - 20
            11x = 11
              x = 1


Rational Equations

    Rational Equation - contains one or more rational expressions; they have fractions with a variable in the denominator 

  ex: 5/2x = 17/18 - 1/3x   - in this example multiply everything by the least common denominator (18x)
        45 = 17x - 6
        51 = 17x 
        17x = 51
        x = 3
  
  ex02: x/x-2 = 2/x-2 - 2/3   - in this example multiply everything by 3(x-2)
        3x = 6 - 2(x-2)
        3x = 6 - 2x + 4
        5x = 10
         x = 2

  Identity - an equation that is true for all real numbers; both sides must be defined

  Conditional Equation - equation that is not an identity but that is true for at least one real number

  Inconsistent Equation - equation that is not true for even one real number the lack of a solution is represented by the empty set symbol
  
  ex: is the equation 4x - 7 = 4(x -1) + 3 an Identity, a Conditional equation, or an Inconsistent equation
      4x - 7 = 4x - 4 + 3
      4x - 7 = 4x - 1         
          -7 = -1               -- Inconsistent
