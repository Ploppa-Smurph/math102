Day 02: 01.02 - Linear and Rational Equations


01.02 Linear and Rational Equations

Linear Equation (in One Variable) - equation can be written as:
     ax + b = 0
    
Root - aka solution

Solution Set - set of all solutions that satisfy equations

Equivalent Equations - two or more equations that have the same solution set

Solving Linear Equations : 
    01 - Simplify algebraic expression by removing grouping symbols and combining like terms
    02 - Collect all variable terms on one side and numbers (constant) terms on the other side
    03 - Isolate the variable and solve
    04 - Check the proposed solution in the original equation

Ex:
a. 4x + 5 = 29
         -5    -5
      4x = 24
       x = 6

b. 4(2x + 1) = 29 + 3(2x - 5)
      8x + 4 = 29 + 6x - 15
      8x + 4 = 6x + 14
      8x = 6x + 10
      2x = 10
      x = 5

Equations are easier to solve if they do not contain fractions. 
    multiply both sides of the equation by the least common denominator of an fraction in the equation

c. (x - 3) / 4 = [(5 / 14) - (x + 5) / 7]   -- denominators of 4, 14, 7 -- LCD = 28
      28((x - 3)/4) = 28[(5/14) - ((x+5)/7)]
      7(x - 3) = 2(5) - 4(x + 5)
      7x - 21 = 10 - 4x - 20
      7x + 4x = -10 + 21
      11x = 11 
      x = 1

Rational Equation - equation that contains one or more rational (fraction) expressions

d. (5 / 2x) = (17 / 18) - (1 / 3x)
      9(5) = 17(x) - 6
      45 + 6 = 17x
      51 = 17x
      x = 3


e. (x / x-2) = (2 / x-2) - (2 / 3)   : LCD = 3(x-2)
      3(x) = 3(2) - 2(x - 2)
      3x = 6 - 2x + 4
      3x = 10 - 2x 
      5x = 10
      x = 2


Identity - equation is true for ALL real numbers, both sides are defined

Conditional Equation - not an identity, but true for at least one real number

Inconsistent Equation - not true for even one real number, represented by 'empty set symbol'

f. 4x - 7 = 4(x - 1) + 3 
      4x - 7 = 4x - 4 + 3 
      4x - 7 = 4x - 1  = inconsistent equation  \


Day 02: 01.04 - Complex Numbers


01.04 Complex Numbers

  Imaginary unit i is defined as i = sqrt(-1), where i^2 = -1
     i is the square root of negative 1, where i squared is equal to negative

ex: sqrt(-25)     square root of -25
       sqrt(-1) * sqrt(25)
       i[sqrt(25)]
       5i       -- would be the final answer simplified

Complex Numbers - set of all numbers in the form a + bi

Real Part - the real number 'a' in 'a + bi'

Imaginary Part - the real number 'b' in 'a + bi'

Pure Imaginary Number - the 'bi' in the 'bi'

Adding and Subtracting Complex Numbers
    Adding: (a + bi) + (c + di) = (a + c) + (b + d)i

    Subtracting: (a + bi) - (c + di) = (a - c) + (b - d)i

ex a: (5 - 2i) + (3 + 3i)
      (5 + 3) + (-2i + 3i)
      8 + i  

ex b: (2 + 6i) - (12 - i)
         (2 + 6i) - 12 + i
         (2 + -12) + (6i + i)
         -10 + 7i 

Multiplication of Complex Numbers - Use FOIL method

ex c: 7i(2 - 9i)
         (7i * 2) (7i * -9i)
         14i - 63i^2    
         14i - 63(- 1)
         14i + 63
         63 + 14i -- correct format [real number 'a', then pure imaginary number 'bi']

ex d: (5 + 4i)(6 - 7i)
         30 - 35i + 24i -28i^2
         30 - 11i - 28i^2  
         30 - 11i - 28(- 1)
         58 - 11i 

Complex Conjugate - complex numbers are the same, but the signs are different
   (a + bi)(a - bi) = a^2 + b^2

** Use 'complex conjugates' to simplify problems with complex numbers in the denominator **

ex e: 3i / 4 + i
         [3i(4 - i)] / [(4 + i)(4 - i)]
         (12i - 3i^2) / (4^2 + 1^2)    ** the i^2 from the 3i^2 can be considered -1
         (12i + 3) / 16 + 1
         (3 + 12i) / 17   -- final solution; aka [3/17 + (12/17)i]

ex f:  (5 + 4i) / (4 - i)
          [(5 + 4i)(4 + i)] / [(4 - i)(4 + i)]
          [20 + 5i + 16i + 4i^2] / [4^2 + 1]
          (20 + 21i - 4) / (16 + 1)    
          (16 + 21i) / 17 -- aka: [16/17 + (21/17)i]

Principal Square Root - defined by: sqrt(-b) = i(sqrt(b))
    CORRECT: 
    sqrt(-25) * sqrt(-4) 
       i(sqrt(25)) * i(sqrt(4))
       5i * 2i = 10i^2
       10(- 1) = -10     -- correct answer

    INCORRECT:
    sqrt(-25) * sqrt(-4)
       sqrt(100)
       10                -- that's wrong!!!

ex g: sqrt(-27) + sqrt(-48)
      i(sqrt(27)) + i(sqrt(48))
      3i(sqrt(3)) + 4i(sqrt(3))
      7i(sqrt(3))  

ex h: [-2 + sqrt(-3)]^2
        [(-2 + sqrt(-3)) * (-2 + sqrt(-3))]  
        [(-2 + i(sqrt(3))) * (-2 + i(sqrt(3)))]   -- then FOIL
        4 - 2i(sqrt(3)) - 2i(sqrt(3)) + i^2(sqrt(9))
        4 - 4i(sqrt(3)) + (-1)(3)
        1 - 4i(sqrt(3)) 

ex i: (-14 + sqrt(-12) / 2)
        [-14 + i(sqrt(12))] / 2
        [-14 + 2i(sqrt(3))] / 2
        -7 + i(sqrt(3))
