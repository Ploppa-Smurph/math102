Ch 04.01 - Exponential Functions

Exponential Functions - any function whose equation contains a variable in the exponent
    f(x) = b^x or y = b^x


** NOT Exponential Functions ** 
    F(x) = x^2    -- variable in base, not exponent
    G(x) = 1^x    -- base must be positive constant, other than 1
    H(x) = (-1)^x -- base must be positive
    J(x) = x^x    -- variable in both base and exponent

    ex: f(x) = 42.2(1.56)^x f(x) = average amount spent in dollars; x = hours spent at the mall
           find average amount spent after 3 hours

        f(3) = 42.2(1.56)^3
        f(3) = $160.208      --- question asks us to round to nearest dollar
        f(3) = $160

    ex02: graph f(x) = 3^x
            x              f(x) = 3^x
           ---------------------------
            -2             1/9   = (-2, .11)
            -1             1/3   = (-1, .33)
            0              1     = (0, 1)
            1              3     = (1, 3) 
            2              9     = (2, 9) 
            
    ex03: graph f(x) = (1/3)^x
           x              f(x) = (1/3)^x
          ------------------------------
          -2           (1/3)^-2 = 3^2 = (-2, 9)    
          -1           (1/3)^-1 = 3^1 = (-1, 3)

Characteristics of Exponential Functions of the Form f(x) = b^x  
1) the Domain of f(x) = b^x consists of all real numbers (-inf, inf). 
    the Range of f(x) = b^x consists of all positive numbers (0, inf) 

2) the graphs of all exponential functions of the form f(x) = b^x pass through the point (0, 1) 
     because f(0) = b^0 = 1, the y-intercept is 1, there is NO x-intercept.

3) If b > 1, f(x) = b^x has a graph that goes up to the right and is an increasing function.
    The greater the value of b the steeper the increase.

4) If 0 < b < 1, f(x) = b^x has a graph that goes down to the right and is a decreasing function.
    The smaller the value of b, the steeper the decrease.

5) f(x) = b^x is one-to-one and has an inverse that is a function.

6) The graph of f(x) = b^x approaches, but DOES NOT touch the x-axis. 
     The x-axis, or y = 0, is a horizontal 'asympotote'.

e
  the irrational number, symbolized by e, appears as the base in many applied exponential functions
 e = 2.718281827; often shown on calculators as e^x

Compound Interest - interest computed on an original investment as well as on any accumulated interest
Principal - amount initially invested
Coumpounded Semiannually - compound interest is paid 2 times a year, every 6 months.
Compounded Quarterly - compound interest is paid 4 times a year, every 3 months.
Continous Compounding - where the number of compounding periods increases infinitely.

Formula for Compound Interest - after t years, the balance, A, in an account with principal, P
     and annual interest rate r (in decimal form) is given by the following formulas:
 1) n compounds per year: A = P(1 + r/n)^nt
 
 2) for continous compounding: A = Pe^rt

 ex: sum of $10,000 is invested at an annual rate of 8%. Find the balance in the account after 5 years subject to: 

 a) quarterly compound: A = 10000(1 + .08/4)^(4*5)
                           A = 10000(1 + .02)^20
                           A = $14859.47

 b) continous compound: A = 10000e^(.08 * 5)
                           A = $14918.82 


04.02 - Logarithmic Functions

      ** NO HORIZONTAL LINE can be drawn that intersects the graph of an exponential function at more than 1 point.
      ** Exponential Functions are 1-to-1 (1:1) and have an inverse.

The inverse function of the exponential function with base b is called the logarithmic function with base b.

Logarithmic Function - For x > 0 & b > 0, b != 1
    y= log_b*x is equivalent to b^y = x
 the function f(x) = log_b*x is the logarithmic function with base b.

 y = log_b*x is the Logarithmic Form
 b^y = x is the Exponential Form

ex: write in equivalent exponential form
   a) 3 = log_7 x
         7^3 = x

   b) 2 = log_b 25
         b^2 = 25

   c) y = log_4 26
         4^y = 26

ex02: write in equivalent logarithmic form
   a) 2^5 = x
         log_2 x = 5

   b) b^3 = 27
         log_b 27 = 3

   c) e^y = 33
         log_e 33 = y

Basic Logarithmic Properties Involving 1
   - log_b b = 1; 1 is the exponent to which b is raised to obtain b

   - log_b 1 = 0; 0 is the exponent to which b is raised to obtain 1

Inverse Properties of Logarithms -- For b > 0 & b != 1

   - (b^log_b x) = x -- b raised to log with base b of a number equals that number
